# 个人博客设计方案

## 1. 定位

- 计算机专业学生个人站点：技术学习笔记 + 学业生活记录
- 首页即个人品牌门面

## 2. 技术栈

| 项 | 方案 |
|---|---|
| 框架 | Astro + Retypeset 主题 |
| 部署 | GitHub Actions 自动部署至 GitHub Pages（`liushenwuzhu-alpaca.github.io`，user pages 根路径） |
| 域名 | 暂用默认域名，后期可购独立域名（.dev / .me） |
| 内容格式 | Markdown / MDX |

## 3. 视觉规范（纸上技术志）

### 3.1 基础色（固定，不随配色方案变化）

| 角色 | 浅色 | 暗色 |
|---|---|---|
| 纸（背景） | `#F4F4EF` | `#1A1C1B` |
| 墨（正文） | `#1B1E1C` | `#E5E4DB` |

### 3.2 配色方案（仅替换强调色即可切换）

| 方案 | 浅色强调色 | 暗色强调色 | 色相特征 |
|---|---|---|---|
| 松烟绿（默认） | `#3A5F52` | `#82A898` | 低饱和深青绿，偏灰 |
| 朱批绛红 | `#8C2F39` | `#D4898F` | 低饱和暗红 |
| 普鲁士蓝 | `#2F4E6B` | `#8FAECC` | 低饱和深蓝，偏灰 |
| 靛蓝墨 | `#4653A0` | `#9AA5E0` | 中饱和蓝紫 |
| 苔金 | `#8A6D2F` | `#C9A85C` | 低饱和暗黄 |
| 烟棕 | `#6B4F3A` | `#B5987A` | 低饱和暖棕 |
| 灰紫藤 | `#5B5466` | `#A79FBC` | 低饱和灰紫 |
| 纯墨 | 无 | 无 | 不设强调色：链接仅靠下划线与字重区分，选区用墨色反白 |

- 方案名称仅为代号，实际颜色以 hex 值为准
- 切换配色只需替换强调色变量，纸、墨、版式均不变
- 暗色版强调色规则：在浅色版基础上提亮、降饱和

### 3.3 使用规则

- 强调色仅用于：链接悬停、引文左边框、文字选区
- 次级信息用降低不透明度表达，不新增颜色
- 字体：标题与正文 Source Serif 4 + Noto Serif SC，正文 17–18px，行高 1.8；代码 JetBrains Mono
- 版式：行宽 ≤ 66ch 居中；禁卡片、阴影、圆角组件、渐变、hero 大图；分隔仅用留白 + 1px 发丝线（12% 不透明度墨色）
- 动效仅三种：链接下划线 300ms 生长、箭头 hover 位移、明暗模式 300ms 过渡

## 4. 信息架构

```
导航：文章 · 归档 · 项目 · 此刻 · 关于（+ 搜索 / 明暗切换）

首页：一句话自我介绍
      → 精选文章（手动指定 3–5 篇）
      → 最近更新（5 篇）
      → 项目（3 个，纯文字列表）

文章页：标题 / 日期 / 分类 / 标签 / 阅读时长
      → TOC → 正文（代码块带复制按钮、文件名标签）
      → 系列导航 → 上一篇 / 下一篇 → Giscus 评论区

归档页：按年份分组时间线
项目页：项目名 + 一句话描述 + →
关于页：你是谁 · 在学什么 · 博客写什么 · 邮箱联系方式
```

## 5. 功能清单

**主题自带（零成本）**：RSS、Sitemap、SEO 元信息、暗色模式、TOC、归档/分类/标签、LaTeX 公式、代码高亮、MDX、阅读时长

**第一批（必装）**：

| 功能 | 方案 |
|---|---|
| 评论 | Giscus（GitHub Discussions 驱动，数据存自有仓库） |
| 全文搜索 | Pagefind（构建时索引，纯静态） |
| 代码块增强 | Expressive Code（复制按钮、文件名、行高亮） |
| 系列文章 | frontmatter series 字段 |
| OG 图自动生成 | satori |

**第二批（后续）**：Umami 访问统计、/now 此刻页、友链页、文章「最后更新于」（git 提交时间）、GitHub 贡献热力图

**禁用清单**：阅读量计数挂件、音乐播放器、看板娘、鼠标特效、页面加载进度条、弹窗订阅
