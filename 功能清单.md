## 需要权衡(暂不做)

- **阅读量计数** — 纯静态做不到，需借 Umami API 或 Giscus reaction 变通
- **PWA 离线** — 静态站收益小，维护成本高
- **多语言** — 框架已支持(`moreLocales`)，但双语内容维护成本高
- **加宽栏位** — 660px 对应每行约 37 汉字，正处最佳行长，加宽伤阅读
- **字号线收敛** — 页标题 26→30px、砍 24/28 两级，涉及全文排版回归，待专门排期
